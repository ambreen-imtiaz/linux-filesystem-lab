# Linux File System Lab

## Overview

This project demonstrates basic Linux file system operations using Kali Linux.

## Objectives

- Create directories
- Create files
- Navigate directories
- Copy and rename files
- Change file permissions
- Search for files
- Understand Linux file system structure

## Commands Used

```bash
mkdir
cd
pwd
ls
touch
cp
mv
chmod
find
tree
```

## Skills Learned

- Linux File Management
- Directory Navigation
- File Permissions
- Basic Command Line Usage
